# OpenHabitat

A manual for automatic longevity, happiness, and performance built into the walls.

> **Read the handbook at [openhabitat.org](https://openhabitat.org).** This repository is the source.

## Ethos

Your caveman biology is outgunned by space-age threats it never evolved to see. Invisible pollutants, bad air, bad water, bad light, and weaponized products cost the average person 2–10 healthy years — and the damage compounds across generations. 

Where you are born should not be your destiny. We can fix the Geographic Lottery by endowing every man, woman, and child with an optimal habitat to grow up in.

OpenHabitat is how we take back control. It's not a supplement stack or a magic pill — it's the principles, measurements, systems, and products behind a habitat that makes longevity, performance, and happiness the automatic default. Every recommendation has been vetted over a decade of personal testing. Follow it blindly, or read the principles and customize to your own Habitat.

## Commands

All commands run from the project root:

| Command          | Action                                       |
| :--------------- | :------------------------------------------- |
| `pnpm install`   | Install dependencies                         |
| `pnpm dev`       | Start local dev server at `localhost:4321`   |
| `pnpm build`     | Build production site to `./dist/`           |
| `pnpm preview`   | Preview the build locally before deploying   |
